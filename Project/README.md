##Project in DD2525

This contains the files for our project on fuzzing in the course DD2525.
It contains several folders. The folders named *pdf_mutators* and *png_mutators* contains the python scripts for the mutators. 
They also contain helper scrips that that can be used to see if the file has been modified. They are ran using *python3 pdfTOBytes.py FILE* or *python3 png_tester.py FILE* where *FILE* is the file to be randomly modified. 

The seeds folders contains the seeds we tried, which are valid files. Notes and the png image is for us. 

The folder *code_LLM* contains the LLM code.
